# 啟動腳本顯示範例

當你執行 `start_v2.bat` 時，CMD 會顯示：

```
============================================
  AI Expert System v2.0
============================================

[OK] Python 已安裝

[0/2] 正在偵測本機 IP...
[OK] 偵測到本機 IP: 192.168.5.3

[1/2] 啟動後端 API Server (Port 8000)...
     等待後端就緒...
[2/2] 啟動前端介面 (Port 8501)...

============================================
  啟動完成！
============================================

  [本機存取]
  前端介面:       http://localhost:8501
  後端 API:       http://localhost:8000
  API 文件:       http://localhost:8000/docs

  [局域網存取] (同網段電腦可使用)
  前端介面:       http://192.168.5.3:8501
  後端 API:       http://192.168.5.3:8000

  頁面功能:
    - 專家問答:   Chat 頁面
    - 管理後台:   Admin 頁面
    - 統計儀表:   Stats 頁面

  [安全提示]
  - 局域網內可存取: 同網段電腦可透過上方 IP 連接
  - 公網已隔離: 外網無法存取 (需確保路由器未開放 Port)

  資料庫: 首次啟動時自動建立
  檔案監控: 後端啟動後自動開啟

  提示: 關閉各服務視窗即可停止對應服務
============================================
```

## 重點特色

1. ✅ **自動偵測 IP**: 不再顯示 `0.0.0.0`，而是顯示實際的局域網 IP (如 `192.168.5.3`)
2. ✅ **分類顯示**: 清楚區分本機存取和局域網存取的 URL
3. ✅ **安全提示**: 提醒使用者確認路由器設定
4. ✅ **支援多種網段**: 自動偵測 192.168.x.x / 10.x.x.x / 172.x.x.x 網段

## IP 偵測邏輯

腳本會依序搜尋：
1. 192.168.x.x 網段 (最常見的家用/辦公室網段)
2. 10.x.x.x 網段 (大型企業網路)
3. 172.16-31.x.x 網段 (中型企業網路)

如果都找不到，則顯示 `localhost` 並提示警告訊息。
