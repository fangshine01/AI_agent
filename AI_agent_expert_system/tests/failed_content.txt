--- 投影片 2 ---
Problem issue
N706 PTST G Line Gap, D Line Gap PTST Yield Loss ~8%(含Total Yield Loss 18%)
Problem description
N706 在A3LR 檢出 In_Film
比對點位為Oven pin
Analysis root cause
BSITO 在Oven 製程中，機台Pin 汙染玻璃，玻璃翻轉後清洗，髒污仍在
Oven 工程師處理流程疏失
檢測修補工程師誤復活已scrap panel
Contentment action
A3LR、A8HR攔檢，針對斷線、M1 residue做rework
異常Oven tool slot關閉
異常Cooling tool slot 關閉
Corrective action
針對檢測修補線上人員再宣導，避免復活 scrap panel
針對Oven 值班工程師再宣導，因N706 無ATST，需針對異常 Glass 做 film rework
Preventive action
Oven Pin 保養週期:  無 >>年保更換、研磨(待驗證pin life time)
異常Cooling tool slot ，開爐保養，回線後異常slot 待首批驗證
異常Oven tool slot關閉，待開爐保養(Oven 年保才開爐)，回線後待首批驗證
